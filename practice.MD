1. Which of the following are operators, and which are values?
```
*
'hello'
-88.8
-
/
+
5
```
Answer: 
```
* - operator
'hello' - value
-88.8 - value
- - operator
/ - operator
+ - operator
5 - value
```

10. Why does this expression cause an error? How can you fix it?
'I have eaten ' + 99 + ' burritos.'
This expression causes an error because it is not possible to concatinate strings and integers. this can be fixed by converting the integer 99 with int(99) or by simply adding quotes around 99 such as '99'.
